# OneWindow CIF & Account Automation Framework (Excel-Driven)

Automation framework for **One Window banking application** using **Selenium**, **Java**, **Cucumber (BDD)**, and **Excel-driven test data**.  
The framework supports **multiple workflows executed sequentially** for different customer information and account creation entries.

---

## Overview

This project showcases my expertise in **enterprise automation and BDD methodology**.  
It enables scalable, maintainable, and **fully automated testing** of One Window CIF and account creation workflows.

Key highlights:

- **Excel-driven multiple flows**: Each row in Excel represents a unique CIF/account workflow
- End-to-end automation without manual intervention
- Reusable step definitions and modular Page Object Model for maintainability
- Demonstrates **professional-grade enterprise automation methodology**
